# SUBIO-TUN

اسکریپت خودکار نصب و مدیریت تانل SSH معکوس (Reverse SOCKS5) با استفاده از **SUBIO-SSH**.
این اسکریپت جایگزین نسخه دستی شده و به صورت تعاملی کانفیگ‌های لازم برای سرور ایران و خارج را ایجاد می‌کند.

## ویژگی‌ها

- **نصب خودکار SUBIO-SSH**: پشتیبانی از Ubuntu/Debian (توسط PPA) و Fedora/RHEL (توسط COPR) و یا کامپایل خودکار از سورس.
- **تولید کلید خودکار**: ایجاد کلید Ed25519 و مدیریت `authorized_keys`.
- **نظارت و مدیریت پیشرفته**: همراه با اسکریپت‌های هوشمند `subio-manager.py` (برای پایداری اتصال) و `subio-lane-guard.py` (برای بازیابی SOCKS).
- **پشتیبانی چند سروره**: امکان اتصال چندین سرور خارج (آلمان، فرانسه، ترکیه و ...) به یک سرور ایران.
- **انتخاب پورت دلخواه**: پورت SOCKS برای هر سایت قابل تنظیم است.

## پیش‌نیازها

- یک سرور در ایران
- یک (یا چند) سرور در خارج
- دسترسی `root` (اجرای دستورات با sudo)
- اوبونتو ۲۰.۰۴+، دبیان ۱۱+ یا سرورهای RHEL/CentOS/Alma 8+

## نحوه نصب و استفاده

۱. ابتدا وارد سرور **ایران** شوید. اسکریپت را دانلود و اجرا کنید:
```bash
chmod +x setup.sh
./setup.sh
```

۲. در مراحل نصب:
- گزینه `iran` را انتخاب کنید.
- آیپی سرور ایران و سرورهای خارج را وارد کنید.
- پورت‌های SOCKS را تعیین کنید.
- در پایان کلید عمومی تولید شده را کپی کنید.

۳. سپس وارد سرور **خارج** شوید. اسکریپت را اجرا کنید:
```bash
chmod +x setup.sh
./setup.sh
```

۴. در مراحل نصب:
- گزینه `kharej` را انتخاب کنید.
- آیپی سرور خارج و ایران را وارد کنید.
- در انتها، کلید عمومی کپی شده از سرور ایران را در `/root/.ssh/authorized_keys` سرور خارج (یا برعکس بسته به جهت) اضافه کنید تا احراز هویت انجام شود. (در آپدیت‌های بعدی این بخش هم خودکار می‌شود).

## مدیریت و مانیتورینگ

اسکریپت چند دستور برای مدیریت تانل اضافه می‌کند:

بررسی وضعیت تانل‌ها و پورت‌ها:
```bash
./setup.sh status
```

تست سرعت و سلامت پورت SOCKS مشخص (مثلا ۱۰۸۱۰):
```bash
./setup.sh test 10810
```

همچنین برای بررسی دقیق‌تر سرعت و اتصال بعد از ایجاد تانل، می‌توانید از دستور زیر استفاده کنید (پورت ۱۰۸۱۰ را با پورت خود جایگزین کنید):
```bash
speed=$(curl -x socks5h://127.0.0.1:10810 -o /dev/null  -L -w '%{speed_download}' https://fsn1-speed.hetzner.com/100MB.bin) && awk -v s="$speed" 'BEGIN { printf "Speed: %.2f MB/s = %.2f Mbps\n", s/1000000, s*8/1000000 }'
```

مشاهده لاگ لحظه‌ای تانل:
```bash
journalctl -u subio-manager -f
```

## نحوه اتصال پنل ثنایی

پس از اینکه هر دو سرور ایران و خارج تنظیم شدند و فرمان `./setup.sh status` نشان داد که پورت متصل است، می‌توانید از SOCKS در پنل ثنایی استفاده کنید:

1. در پنل ثنایی به بخش **تنظیمات / Outbounds** بروید.
2. یک خروجی جدید از نوع `socks` ایجاد کنید.
3. آدرس را `127.0.0.1` و پورت را پورتی که در تنظیمات انتخاب کردید (مثلا `10810`) وارد کنید.
4. آن را به کانفیگ ورودی خود (Inbound) متصل کنید.
